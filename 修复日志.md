## 技术说明

### Element Plus 图标命名规则：
- Vue 3 推荐使用 kebab-case
- Element Plus 图标支持 kebab-case 和 PascalCase
- 统一使用 kebab-case 保持一致性

### CORS 原理：
- `allow_origins=["*"]` 允许所有来源
- 开发环境可使用通配符，生产环境建议限制为具体域名
- 内网访问通过 `192.168.x.x` IP 访问，属于允许范围
